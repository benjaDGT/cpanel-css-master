Zapier

Zapier allows you to connect different software apps to IDXBoost.
Get data into/out of your account or perform advanced actions in IDXBoost,
such as having an appointment add a tag to a contact.

Connect Software Apps to IDXBoost

  • IDXBoost

  • Zapier

  • Other Software

You can create your Zap App to feed the IDX Boost CRM or for the IDX Boost CRM to feed another Zap App.
Currently, our Zap App is in development mode until it's approved for production by Zapier, it can only be
used by invitation for beta users.

IDX Boost Zapier App Authorization

In order to use the application, during one of the connection processes, you will need to give access to the IDX Boost CRM.

Actions

An action push or put data into your IDX Boost CRM Zap App from their Zaps

Create Lead & Create Contact Inquire These actions add a new lead or an inquire into the IDXBoost CRM. if an email address matches with a lead already in the system, it will add the inquiry below their profile. If it's a new lead, the system will create it as Hot Lead.

Triggers

New Lead This event sends IDXBoost to Zapier when it feeds a lead with the following adjustment

New Contact Inquiry This event sends IDX Boost to Zapier when it feeds a form with the following adjustment

New Property Inquiry This event sends IDX Boost to Zapier when it feeds a property form with the following adjustment
